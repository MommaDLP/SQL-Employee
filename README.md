# SQL-Employee 

A command-line app for managing employee data using Node.js, PostgresSQL and Inquirer.

## Installation
Run 'npm i' to install <br>
Next run 'node index.js' to start the app

## Features
-'All Departments',
-'All Roles',
-'All Employees',
-'Add Department',
-'Add Role',
-'Add Employee',
-'Update Employee Role',
-'Exit'
